#  General Knowledge Quiz (Python)

The **General Knowledge Quiz** is a console-based Python application designed to test a user's basic general knowledge through a series of interactive questions. The program provides an engaging quiz experience by accepting user responses, validating answers, calculating scores, and displaying a detailed performance report at the end.

This project demonstrates the practical implementation of fundamental Python programming concepts such as functions, loops, conditional statements, lists, string manipulation, input validation, and score tracking. The code is organized into reusable functions, making it easy to understand, maintain, and extend with additional questions or features.

##  Features

* Interactive command-line quiz interface
* User name input with validation
* Clear quiz instructions before starting
* Countdown before the quiz begins
* Three general knowledge questions
* Case-insensitive answer checking
* Automatic removal of extra spaces from user input
* Instant feedback after every answer
* Real-time score updates
* Final score and percentage calculation
* Grade assignment based on performance
* Personalized performance remarks
* Quiz summary report
* Option to play the quiz multiple times without restarting the program

##  Python Concepts Used

* Variables
* Functions
* Lists
* Loops
* If-Else Statements
* String Methods
* Input Validation
* Score Tracking
* f-Strings

##  How It Works

1. The program welcomes the user and displays the quiz rules.
2. The user enters their name before starting the quiz.
3. A short countdown is displayed before the questions begin.
4. The user answers three general knowledge questions.
5. Each answer is checked against the correct response.
6. One point is awarded for every correct answer.
7. After all questions are completed, the program calculates the total score, percentage, and grade.
8. A detailed performance report and quiz summary are displayed.
9. The user can choose to play the quiz again or exit the application.
